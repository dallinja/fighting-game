# fighting-game

A javascript "Street Fighter" style game with sprites.

Clone the project, and then in the project's root directory run the following to open the game in your default browser and play:

```bash
open ./index.html
```

Player 1 controls:

- Left: A
- Right: D
- Jump: W
- Attack: Spacebar

Player 2 controls:

- Left: Left Arrow
- Right: Right Arrow
- Jump: Up Arrow
- Attack: Down Arrow
